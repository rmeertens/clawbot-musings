About This Map

This page fetches the latest tweets from verified world leaders and analyzes their sentiment using machine learning. The map updates every hour to show the current mood of global leadership.

How it works:

  • Identifies the top representative (PM, President, or monarch) for each country
  • Fetches their latest tweets using the Twitter API
  • Analyzes tweet sentiment (very positive, positive, neutral, negative, very negative)
  • Visualizes results on an interactive world map

Data sources:

  • World leaders list: Curated dataset
  • Tweets: Twitter API (10 most recent tweets per leader)
  • Sentiment analysis: Local LLM (on-device, no external API)
  • Map data: Natural Earth via GeoJSON
